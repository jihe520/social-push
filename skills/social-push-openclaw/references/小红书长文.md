## 上传长文 workflow (OpenClaw)

1. 打开发布长文页面：`browser(action=navigate, targetUrl="https://creator.xiaohongshu.com/publish/publish?source=official&from=tab_switch&target=article")`
2. 查看交互元素：`browser(action=snapshot, refs="aria")`
3. 进入长文模式：`browser(action=act, request={kind:click, ref:"新的创作ref"})`
4. 填写标题：`browser(action=act, request={kind:fill, ref:"标题输入框ref", text:"{标题}"})`
5. 点击正文编辑框：`browser(action=act, request={kind:click, selector:".ProseMirror"})`
6. 输入正文内容：`browser(action=act, request={kind:type, selector:".ProseMirror", text:"{正文内容}"})`
7. 一键排版：`browser(action=act, request={kind:click, ref:"一键排版ref"})`
8. 查看状态：`browser(action=snapshot, refs="aria")`
9. 下一步：`browser(action=act, request={kind:click, ref:"下一步ref"})`
10. 查看发布设置：`browser(action=snapshot, refs="aria")`
11. 输入简介：`browser(action=act, request={kind:type, selector:".ProseMirror", text:"{简介内容}"})`
12. 添加话题：`browser(action=act, request={kind:type, selector:".ProseMirror", text:" #{话题}"})` → `browser(action=act, request={kind:press, key:"Enter"})`
13. 暂存草稿：`browser(action=act, request={kind:click, ref:"暂存离开ref"})`

## 图片上传

粘贴 Markdown 内容后，替换 `![alt](path)` 图片引用：

1. 用 JS 找到图片引用段落：
   ```
   browser(action=act, request={kind:evaluate, fn:"const ps = document.querySelectorAll('.ProseMirror p'); for (let i = 0; i < ps.length; i++) { if (ps[i].innerText.startsWith('![')) { const range = document.createRange(); range.selectNodeContents(ps[i]); const sel = window.getSelection(); sel.removeAllRanges(); sel.addRange(range); break; } }"})
   ```
2. 删除选中文本：`browser(action=act, request={kind:press, key:"Backspace"})`
3. 上传真实图片替代

## 注意事项

- 每步操作后用 `browser(action=snapshot)` 确认元素 ref
- 正文编辑器为 ProseMirror，使用 `.ProseMirror` 选择器
